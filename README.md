# Chess4AI
一个突发奇想简单测试AI能力的项目，连接到大模型，通过传输字符串化的棋盘和走棋步骤来和大模型打谱对弈。目前只有国际象棋和中国象棋。
国际象棋使用SAN谱
中国象棋使用UCCI谱

```bash
git clone https://github.com/Ksdb104/Chess4AI
npm install
npm run dev
```
